# Chatbot de Automatización Simple

Asistente virtual que consulta precios de acciones y clima en tiempo real usando web scraping.

## Instalación

```bash
pip install -r requirements.txt
```

## Uso

```bash
python main.py
```

Ejemplos:
- "clima en Colima"
- "precio accion Apple"
- "temperatura en Guadalajara"

## Requisitos

- Python 3.x
- Chrome instalado
- Conexión a internet
